**Positive Points**:

1. O simulador é uma ferramenta útil para planejamento de aposentadoria.
2. É fácil de usar e entender.
3. Oferece várias opções de aposentadoria e regras de transição.
4. Permite a inclusão de dados de outros regimes previdenciários.
5. É uma ferramenta importante para tomada de decisão sobre a aposentadoria.

**Negative Points**:

1. Não considera os vínculos do RGPS.
2. Não permite a inclusão de período de CTC (RGPS) ainda não averbado.
3. Não apresenta claramente as informações salariais das simulações de aposentadoria.
4. Não explica claramente a diferença entre os cenários de simulação.
5. Não apresenta o valor do benefício financeiro atualizado para os servidores que migraram de regime.

**Suggestions**:

1. Incluir a opção de informar manualmente o período de RGPS para simulação correta.
2. Abrir campos para incluir tempo de RGPS também.
3. Informar quem tem o real direito de usufruir da paridade, informando o tempo de Serviço.
4. Explicar o motivo pelo qual a remuneração não aparece na simulação.
5. Incluir a opção de inserção/migração de dados de outros regimes previdenciários para compor todas as contribuições já realizadas pelo servidor.
