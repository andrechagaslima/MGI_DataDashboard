**Positive Points**:

1. Fácil de utilizar
2. Intuitivo
3. Simples
4. Prático
5. Bom para servidores aposentáveis ou próximos de se aposentar
6. Muito útil
7. Excelente ferramenta
8. Fácil de manusear
9. Muito bom
10. Simples e fácil de usar

**Negative Points**:

1. Pode ser complicado para alguns usuários
2. Linguagem técnica pode ser difícil de entender
3. Muitas opções para escolher, mas não é claro qual escolher
4. Não é claro em qual categoria cada servidor se enquadra
5. Resultados não são claros sobre o enquadramento e os pedágios aplicados
6. Não é possível entender a mensagem de não possui parcelas suficientes
7. Não é possível abrir uma possível conclusão de Férias com valores de referência
8. Dados do NIS não estão computados
9. Não é possível exibir de forma clara a previsão de valor da aposentadoria
10. Não é possível entender a mensagem de não atender um dos requisitos

**Suggestions**:

1. Destacar mais os arquivos para download
2. Melhorar a linguagem técnica para torná-la mais acessível
3. Criar um tutorial explicando os diferentes tipos de aposentadorias
4. Criar um chat para ajudar a entender as regras
5. Criar um questionário para aplicar a melhor regra ao caso concreto
6. Acrescentar uma funcionalidade para exibir de forma clara a previsão de valor da aposentadoria
7. Melhorar a clareza sobre o enquadramento e os pedágios aplicados
8. Melhorar a linguagem para torná-la menos técnica
9. Criar um curso tutorial online com orientações básicas
10. Melhorar a integração com os dados do NIS.
