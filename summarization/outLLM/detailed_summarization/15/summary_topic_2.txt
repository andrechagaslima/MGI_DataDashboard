**Positive Points**:

1. Fácil de usar
2. Excelente layout
3. Informação sobre tempo total de contribuição
4. Possibilidade de calcular valor da aposentadoria
5. Simulação de aposentadoria é útil para tomada de decisão

**Negative Points**:

1. Não contempla pessoas com deficiência (PCD)
2. Não permite inclusão de tempo de contribuição do RGPS
3. Não permite inclusão de vínculos do RGPS
4. Não permite inclusão de período de CTC (RGPS) ainda não averbado
5. Não considera vínculos do RGPS
6. Não permite inclusão de tempo de contribuição do RGPS para simulação
7. Não permite inclusão de período de CTC (RGPS) ainda não averbado para simulação
8. Não é claro e fácil de entender
9. Resposta é vaga e não atende às necessidades do usuário
10. Não é útil para servidores com deficiência

**Suggestions**:

1. Incluir opção para incluir tempo de contribuição do RGPS
2. Incluir opção para incluir vínculos do RGPS
3. Incluir opção para incluir período de CTC (RGPS) ainda não averbado
4. Incluir opção para incluir tempo de contribuição do RGPS para simulação
5. Incluir opção para incluir período de CTC (RGPS) ainda não averbado para simulação
6. Melhorar a clareza e a facilidade de entender
7. Melhorar a resposta para atender às necessidades do usuário
8. Incluir opção para incluir tempo de contribuição do RGPS para servidores com deficiência
9. Incluir opção para incluir vínculos do RGPS para servidores com deficiência
10. Incluir opção para incluir período de CTC (RGPS) ainda não averbado para servidores com deficiência.
